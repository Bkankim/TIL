## 두 수의 곱

### 기본 풀이

```python
def solution(num1, num2):
    return num1 * num2
lambda 함수로 한 줄 풀이
python
복사
편집
solution = lambda num1, num2: num1 * num2
배열 두 배 만들기
기본 풀이 (for문)
python
복사
편집
def solution(numbers):
    answer = []
    for i in numbers:
        answer.append(i * 2)
    return answer
리스트 컴프리헨션
python
복사
편집
def solution(numbers):
    return [i * 2 for i in numbers]
lambda 표현식
python
복사
편집
solution = lambda numbers: [i * 2 for i in numbers]
꼬리 문자열
기본 풀이 (for문)
python
복사
편집
def solution(str_list, ex):
    answer = ''
    for i in str_list:
        if ex not in i:
            answer += i
    return answer
리스트 컴프리헨션 + join
python
복사
편집
def solution(str_list, ex):
    return ''.join([i for i in str_list if ex not in i])
lambda 표현식
python
복사
편집
solution = lambda str_list, ex: ''.join(i for i in str_list if ex not in i)
마무리
하루 3문제씩 파이썬 기초 문법을 다양한 방법으로 반복 훈련하면서 코딩 감각을 기르고 있다.
동일한 문제를 다양한 방식으로 풀면서 어떤 풀이가 가장 직관적이고 효율적인지를 고민 중이다.

단순 for문 → 리스트 컴프리헨션 → lambda 표현식까지, 문제 난이도와 목적에 따라 다양한 방법을 선택할 수 있도록 연습하고 있다.
